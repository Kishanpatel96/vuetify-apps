# Vuetify-Todo

## Description
> This is simple a Todo Application.

> In this application you can add your task and delete your task.

> This is mobile and web friendly.

## Languages Used
  - Vuejs (Version 2).
  - CSS
  - HTML
  
### Compiles and hot-reloads for development
```
npm run serve
```
